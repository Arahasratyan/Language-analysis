# Language and Sentiment Analysis Toolkit

A Python tool designed to parse text data across multiple sources, analyze linguistic composition (Armenian, Russian, and English alphabets/words), and visualize word set frequencies over time.

## Datasets

Due to size limitations, the raw source CSV datasets (`source_1.csv` through `source_4.csv`) are hosted externally. 

To run the scripts, download the files from the link below and place them into a folder named `data/` in the project root:

👉 [Download Datasets from Google Drive](https://drive.google.com/drive/folders/1J5sX_Ve3LFNK_G5JqHeYXHZc2VQKZjMs?usp=drive_link)
